## UserService 
Сервис для работы с пользователями, отвечает за регистрацию/авторизацию пользователей.

Функции:

- Регистрация новых пользователей.
- Аутентификация (логин/выход из системы).
- Проверка ролей (USER/ADMIN).
- Выдача и валидация access token.

## BookService 
Сервис для работы с книгами.

Функции:

- CRUD операции с книгами.
- Управление инвентарем (остатки на складе).
- Поиск и фильтрация книг по различным параметрам.
- Управление ценами и скидками.

## AuthorService
Сервис для работы с авторами.

Функции:

- CRUD операции с авторами.
- Поиск авторов по имени, фамилии, ID. 
- *Примечание: авторы не являются пользователями системы.*

## OrderService

Сервис для работы с заказами.

Функции:

- CRUD операции с заказами.
- Поиск и фильтрация заказов.
- Управление корзиной покупок.
- Отслеживание статуса заказа.
- Расчет стоимости заказа с учетом скидок.
- Формирование истории заказов пользователя.
