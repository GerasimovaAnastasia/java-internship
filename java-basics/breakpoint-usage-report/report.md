### Отчёт об использовании отладки и точки останова.

Логическая ошибка: В строке 30 изменила знак сравнения рядом стоящих значений в массиве, что привело к тому, 
что массив сортировался по убыванию, а не по возрастанию.

if (sortedArray[j] < sortedArray[j + 1]) 


Поставила breakpoints на строки: 30, 32, 33, 34, что позволило мне просмотреть переменные temp, sortedArray[j] и sortedArray[j+1]. 
Заметила, что сравнение происходит в неправильную сторону. 

Для устранения ошибки изменила знак на противоположный.

if (sortedArray[j] > sortedArray[j + 1])
